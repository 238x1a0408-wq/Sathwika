## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Sathvika.git
```

2. Navigate to the project folder

```bash
cd Sathvika
```

3. Install dependencies (if applicable)

```bash
npm install
```

4. Run the project

```bash
npm start
```
